# Student Guide

1. Choose a baseline design and run it.
2. Enter Position, Chord, and Twist for each marked blade section.
3. Choose the closest airfoil family: flat foam board, cambered plate,
   symmetric airfoil, or high-lift airfoil.
4. Check the cutting preview and airfoil result before building the blade.
5. Write a hypothesis.
6. Change only one input, one blade section, or one airfoil choice.
7. Compare RPM, torque, electrical mW, Cp, lift/drag, and the performance curves.
8. Export the result and record the measured voltage, current, and mW.
9. Repeat until you can explain why the result changed.

The score is a comparison aid, not proof that a blade is safe or optimal.
